# OpenSuite — Roadmap

## Vision

Build the most comprehensive open-source cross-platform productivity suite, rivaling commercial offerings while remaining free, privacy-respecting, and community-driven.

## Phase 1: Foundation (Sprint 1) ✅
- Core architecture and shared packages
- Notes module
- File Manager module
- Text/Markdown Editor
- CI/CD and Docker infrastructure

## Phase 2: Document Editing (Sprints 2-3)
- Rich document editor (DOCX, ODT, RTF)
- Spreadsheet engine (XLSX, CSV, ODS)
- Formula system
- Charts

## Phase 3: Presentations & PDF (Sprints 4-5)
- Presentation editor (PPTX, ODP)
- Full PDF suite (view, annotate, merge, split, sign)

## Phase 4: Media & Polish (Sprints 6-7)
- Image viewer/editor
- Accessibility audit (WCAG AA)
- Performance optimization
- Full localization (10+ languages)
- Version history

## Phase 5: Collaboration (Future)
- Cloud sync via Cloudflare Workers
- Real-time collaboration
- Sharing and permissions
- Team workspaces

## Phase 6: Advanced Features (Future)
- Rust FFI for performance-critical modules
- Plugin system for extensions
- API for third-party integrations
- Template marketplace
- AI-assisted editing
